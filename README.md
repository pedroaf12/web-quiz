Quiz Web - Demonstração (MVP)
Sobre o projeto

Este projeto consiste em uma demonstração de um quiz desenvolvido utilizando apenas HTML, CSS e JavaScript, sem dependências externas.

O objetivo desta aplicação é servir como um protótipo funcional (MVP) para apresentar a estrutura básica de um sistema de perguntas e respostas, demonstrando a lógica de navegação entre questões, validação das respostas e cálculo da pontuação.

Este projeto não representa a versão final do sistema solicitado, mas sim uma prova de conceito desenvolvida para fins de avaliação técnica

Esse jogo possui:

✅ Três perguntas de exemplo.
✅ 4 alternativas por pergunta.
✅ Validação automática das respostas
✅ Destaque em verde para a resposta correta.
✅ Destaque em vermelho para a resposta errada.
✅ Contador de pontuação.
✅ Botão para jogar novamente ao final.

Tecnologias utilizadas
  HTML5
  CSS3
  JavaScript (Vanilla JS)

  Observação

Este projeto foi desenvolvido exclusivamente como uma demonstração técnica para fins de avaliação.
